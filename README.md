# AI Capstone Project

A Node.js application built with modern JavaScript practices and AI-assisted development workflows.

This repository is part of the AI-Assisted Development capstone track. It demonstrates professional software development using tools such as Cursor, Git, and GitHub.

## Tech Stack

- Node.js (LTS)
- JavaScript
- Git and GitHub
- Cursor AI IDE

## Project Goals

- Learn AI-assisted software development workflows
- Practice using AI tools for coding and documentation
- Follow Git best practices with Conventional Commits
- Build and maintain a professional software project structure

## Development Setup

### Prerequisites

Make sure you have installed:

- Node.js LTS
- Git
- Cursor IDE

### Installation

Clone the repository:

```bash
git clone https://github.com/Ramalsasanka/AI-Capstone.git
cd AI-Capstone
```

Install dependencies (once `package.json` is added):

```bash
npm install
```

### Running the Project

Start the application (once available):

```bash
npm start
```
